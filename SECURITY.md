# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < v0.1  | :x:                |

## Reporting a Vulnerability

UGHI places paramount importance on security, specifically the WASM Capability Token isolation and agent memory sandboxing. 

If you discover a security vulnerability, **PLEASE DO NOT open a public issue.**
Instead, email the core security team directly at **security@ughi.dev**.

We will acknowledge your report within 24 hours and aim to resolve the issue with a patch within 48 hours for critical CVEs.
